# QMAX Web

Landing page desarrollada para validar el interés comercial en motos eléctricas QMAX en Bolivia.

## Descripción

Proyecto web enfocado en la promoción de preventa de motos eléctricas, captación de clientes potenciales y validación de demanda mediante formulario de registro.

## Funcionalidades principales

- Landing page informativa para motos eléctricas QMAX.
- Catálogo de modelos referenciales.
- Formulario de registro de interés.
- Captación de datos como ciudad, modelo preferido y uso principal.
- Conexión con base de datos MySQL para almacenamiento de leads.
- Botones de contacto por WhatsApp y redes sociales.

## Tecnologías utilizadas

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript

## Estado del proyecto

Proyecto personal desarrollado como prototipo para validación comercial.
